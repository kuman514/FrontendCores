# FrontendCores
프론트엔드 개발자 면접에 대비할 수 있는, 그와 동시에 프론트엔드의 각종 기술에 대한 이해를 좀 더 확고하게 익힐 목적으로 개설된 레포지토리입니다. 저 자신의 능률 향상을 위해 만들었지만, 혹시 보시게 될 여러분께도 도움이 된다면 저도 기쁠 것입니다.

## 주의사항
제가 알고 있는 지식이 틀렸을 수도 있기 때문에, 항상 교차 검증 후 갱신을 진행하고 있습니다.
